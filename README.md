# 💰 Sistema Bancário BBSC

Sistema de simulação bancária desenvolvido em Java com o objetivo de praticar os conceitos de Programação Orientada a Objetos (POO).

---

## 🧠 Sobre o Projeto

Este projeto simula um sistema bancário simples com as seguintes funcionalidades:

- Criar contas
- Realizar login
- Simular operações bancárias
- Interface de linha de comando interativa

> Projeto acadêmico desenvolvido para a disciplina de **Programação Orientada a Objetos** (FECAF).

---

## 🚀 Funcionalidades

- 📥 Cadastro de novos clientes
- 🔐 Login com verificação de dados
- 💸 Depósito e saque (simulado)
- 📄 Consulta de informações da conta

---

## 🛠️ Tecnologias Utilizadas

- Java 21 (JDK)
- IntelliJ IDEA
- Programação Orientada a Objetos (POO)
